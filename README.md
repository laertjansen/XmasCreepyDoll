For this project I used an Arduino Uno, an L293D motor driver, a DC motor, a PIR sensor, LEGO bricks for the base, LEDs, and a bit of creativity.

When the PIR sensor detects motion, it triggers the motors to rotate at a very low frequency, creating a subtle vibration.
